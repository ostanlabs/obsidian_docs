# Best Practices

Tips and patterns for effective project management.

## Planning

- Start with milestones
- Break down into stories
- Add tasks incrementally
- Document decisions

## AI Collaboration

- Use natural language
- Ask for analysis
- Review visually
- Iterate quickly

## Visual Review

- Refresh canvas regularly
- Check dependency chains
- Verify structure
- Share with team

## Maintenance

- Archive completed work
- Update status regularly
- Keep relationships clean
- Document changes
