# Common Workflows

Collection of common workflow patterns.

## Daily Workflows

### Morning Standup
```
What did I complete yesterday?
What's in progress?
What should I work on today?
```

### End of Day
```
Mark completed tasks as Complete
What's left for tomorrow?
```

## Weekly Workflows

### Sprint Planning
```
List all NotStarted stories
Prioritize top 5 stories
Create tasks for selected stories
```

### Sprint Review
```
What was completed this week?
What's the milestone progress?
What's at risk?
```

## Project Workflows

### New Feature
```
Create feature F-xxx
Create stories to implement it
Document the design
Link everything together
```

### Decision Making
```
Create decision DEC-xxx
Document context and options
Link to affected entities
Update status when decided
```
