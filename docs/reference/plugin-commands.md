# Plugin Commands

Reference for all Obsidian plugin commands.

## Available Commands

### Populate from Vault
Load all entities onto canvas.

### Reposition Nodes
Auto-arrange in hierarchical layout.

### Refresh Canvas
Reload and update display.

### Clear Canvas
Remove all nodes from canvas.

### Export Canvas
Export canvas as image or JSON.
